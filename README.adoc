= TD 4 - Maven

== Téléchargement

. Lancer `git clone https://github.com/IUT-Blagnac/r2-03-td4-maven-JamesyIO.git`
. Importer le dossier dans un IDE

== Compilation

. Lancer `mvn clean`
. Lancer `mvn package`


== Execution

. Lancer `java -jar target/tp_qualite-1.0.jar`
